# Threema Electron Client

## To Use

```bash
# Clone this repository
git clone https://github.com/ethanrakoff/Threema-BETA-ElectronClient.git
# Go into the repository
cd Threema-BETA-ElectronClient
# Install dependencies
npm install
# Run the app
npm start
```
